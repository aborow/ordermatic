OMC -79 : Update Inventory Details

Update Inventory Details module will update product quantity in inventory details.

In below image you can find format in which you have to made CSV formated file.

Go to Inventory -> Operations -> Inventory Adjustments and click that menu.in records of that menu you can find two options to update quantity.

When you will click to this "Set quantities to 100" it will set quantities of every product to 100.

When you will click this second option "Update quantities based on file" this wizard will open, upload your CSV file here and click Update Inventory Details.

After update, if all products & locations from CSV file will found in system then it will update that line's Product Quantity.

After update if any product will not found in system then it will show this warning.

After update if any location will not found in system then it will show this warning.

After update if there are no lines in inventory details then it will show this warning.
