OMC -64 : Custom BOM Import

Custom BOM Import module will import BOMs in system.

In below image you can find format in which you have to made CSV formated file.

Go to Manufacturing -> Master Data -> Import BOM and click that menu.

When you will click that menu this wizard will open, upload your CSV file here and click IMPORT.

After import, if all products from CSV file will found in system then BOMs will be import successfully.

After clicking import if any product will not found in system then it will show this warning.
