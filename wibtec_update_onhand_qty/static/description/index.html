Update Onhand Quantity provides wizard to update qty onhand based on file.

In below image you can find format in which you have to made CSV formated file.

In Inventory-> Master Data -> Update Onhand Quantity and click that menu.

When you will click that menu this wizard will open, upload you CSV file here and click update onhand quantity.

Once all products and locations will found in system then it will create invetory based on locations and update its lines based on sheet.
