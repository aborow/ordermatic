OMC-80 : Product Category Update Script

"Product Update Category" module provides easy way to add category to product based on list.

In below image you can find format in which you have to made CSV formated file.

Go to Sales -> Products -> Update Product Category and click that menu.

When you will click that menu this wizard will open, choose option Product Category and upload your CSV file here and click update.

After update, if all products & its categories from CSV file will found in system then it will update that product's category to particular product.

After update if any product will not found in system then it will show this warning.

After update if any category of product will not found in system then it will show this warning.

OMC -82 : UOM Update Script

Update UOM will combine UOMs based on uploaded CSV file.

In below image you can find format in which you have to made CSV formated file.

When you will click that menu(mentioned above) this wizard will open, choose option UOM and upload your CSV file here and click update.

Here in below figure UOM for this product is "EAC".

After Update UOM "EAC" will be replaced with "EA".

If any product will not have this UOM then this warning will raise.
