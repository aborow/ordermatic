OMC -63 : Product Archive Script

Product Archive module provides easy way to active products based on list.

In below image you can find format in which you have to made CSV formated file.

Below figure is showing all proucts.

Below figure is showing all BOMs.

Go to Manufacturing -> Master Data -> Update Archive Product and click that menu.

When you will click that menu this wizard will open, upload you CSV file here and click update.

After update, if all products from CSV file will found in system then it will archive products which are not in that csv file but available in system.

After update, if all products & its BOM from CSV file will found in system then it will archive that product's BOM which are not in that csv file but available in system.

After update if any product will not found in system then it will show this warning.

After update if any BOM will not found in system then it will show this warning.
